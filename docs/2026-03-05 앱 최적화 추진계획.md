## 1. 인앱 TTS 상세 (index.js)

### 1.1 입력 방식 4종류 (모두 동일한 수동 버튼 흐름)

| # | 입력 방식 | 트리거 | 핵심 함수 | API |
|---|-----------|--------|-----------|-----|
| 1 | 촬영 | shootBtn | capturePhoto → processImage | generateDescriptionStream |
| 2 | 이미지 업로드 | uploadInput | handleFileSelect → processImage | generateDescriptionStream |
| 3 | 음성 입력 | micBtn | handleMicButtonClick → processTextQuery | generateTextStream |
| 4 | 음성 추가 질문 | detailMicBtn | handleDetailMicClick → processTextQuery | generateTextStream |

### 1.2 공통 TTS 흐름 (4곳 동일)

```
AI 응답 → queueForSpeech() (큐에만 추가, 자동재생 없음)
       → updateAudioButton('play') (▶ 버튼 표시)
       → 사용자 탭 → onAudioBtnClick() → speakNext() → 재생
```

### 1.3 코드 위치 (processImage / processTextQuery 동일 패턴)

- queueForSpeech: 2591/2602 (processImage), 2817/2828 (processTextQuery)
- updateAudioButton('play'): 2607 (processImage), 2832 (processTextQuery)
- onAudioBtnClick → speakNext: 4060줄
